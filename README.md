# robust-anomaly-detection-cbc
Implementation of a modified Stahel-Donoho estimator for outlier detection in skewed CBC data.
**The Problem:** Standard anomaly detection (like Density Estimation) fails on skewed biological data. <br>
**The Solution:** Using the Stahel-Donoho estimator with the Medcouple statistic to create "asymmetric whiskers" for better outlier detection.
